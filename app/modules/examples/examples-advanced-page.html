A collection of advanced examples.

You can run queries to fetch data load on page load.
{{user.first_name}} {{ user.last_name }}
You can run queries to fetch data load on page load using query parameters from the URL.
{{user2.first_name}} {{ user2.last_name }}
You can run queries to fetch data load on page load using path parameters from the URL.
{{user3.first_name}} {{ user3.last_name }}
You can run a lookup function to fetch just-in-time data.
{{ getNameForID(object) }}
You can run assignments in the template to store temporary variables.
My template assigned var is: {{ object.myloop }}
You can execute expressions via JS
{{print('hi')}}
Vue templates are strictly scoped, but you can access things on the Vue prototype.
...
{{ console }} This won't work because console doesn't exist in this scope.
But this will work because it was globally defined on the Vue prototype. {{ my_global_console }} {{ my_global_console.info("HI!!!!!") }}
You can load components recursively.
